## 🚀 Instalação

Você tem duas opções para instalar o repositório no CloudStream:

### Opção 1: Shortcode (Recomendado)
Escreva o seguinte shortcode diretamente no aplicativo:
```text
lietrepo
```

### Opção 2: URL Direta
Copie a URL abaixo e cole no CloudStream:
```http
https://raw.githubusercontent.com/lawlietbr/lietrepo/main/builds/repo.json
```
