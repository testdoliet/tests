O Liet Repo é um repositório de extensões para o CloudStream, focado em trazer fontes brasileiras de conteúdo para o aplicativo. Nossa missão é facilitar o acesso a filmes, séries e animes em português, sempre respeitando os limites legais.

<img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Books.png" alt="Books" width="20" height="20" /> **Características:**
- 📺 Fontes nacionais de streaming
- 🎬 Catálogo atualizado regularmente
- 🇧🇷 Conteúdo dublado e legendado
- 🔄 Sincronização automática com o CloudStream
